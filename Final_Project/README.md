# NSBB 552 Final Project

This repository contains the final data analysis project for NSBB 552, completed in June 2026 by Kathleen Escobar Acuña.

## Project Overview
This project focuses on executing data visualization workflows and pathway enrichment analyses. It demonstrates the application of Python-based tools to process and interpret complex biological datasets.
